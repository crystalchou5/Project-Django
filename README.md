## 專案簡介
##### 這是一個基Django框架開發的店電子商務網站，主要為用戶提供便捷的線上購物體驗。

##### 專案包括了完整的用戶認證系統、購物車、訂單管理等核心功能，並加入顧客的意見反饋功能，提升用戶互動。
